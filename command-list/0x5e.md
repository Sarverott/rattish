# RELEASE PRESSED KEYS
##### BASIC namespace
- hexadecimal: __0x5e__
- octal: __136__
- decimal: __94__
- ASCII character: `^`
  > release all keys
  > example of Ctrl+Alt+Del:
  ```
  _17
  .100
  _18
  .100
  _46
  .300
  ^
  ```
