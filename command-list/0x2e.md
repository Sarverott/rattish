# WAIT TIME PAUSE
##### BASIC namespace
- hexadecimal: __0x2e__
- octal: __056__
- decimal: __46__
- ASCII character: `.`
  > wait argumented amount of milliseconds, expressed as integer.
  > example of waiting 1 second:
  ```
  .1000
  ```
